## Thesis (one line)

PIQI becomes a **layered-light dark editorial system where the red is the only voice raised** — the signature gesture is a branded red→navy→ink route-transition wipe plus a living `@property`-animated hero overlay, executed pure-CSS-first, with `feTurbulence` grain so every surface reads as material. Award-tier craft achieved by perfecting what PIQI already owns, not by bolting on a 3D scene the brand has no assets for.

## Phased rollout (summary)

- **R0 — Design-system foundation.** Token layer (motion, type, space, color ladder, grain/glow) added to `globals.css` as pure additive CSS via the aliasing shim. Zero structural change.
- **R1 — Global chrome + homepage.** Header/GroupNav/MobileNav/Footer + homepage section rhythm, animated overlay angle, clip-path headline reveal, asymmetric vertical grid.
- **R2 — Vertical hubs.** Shared signature hero + scroll-reveal system across all six sibling hubs; "More from PIQI Group" lateral sibling strip.
- **R3 — Consulting spokes.** The only depth-2 pages: intra-hub local nav rail + Consulting-only flyout (mirrors the octopus asymmetry).
- **R4 — About/Contact + interactive polish.** Group-page editorial treatment, magnetic cursor/buttons, route-transition curtain; repair `revolver-heading` (it ships dead Tailwind + a red-rule-violating blue).
- **R5 — Performance/a11y hardening.** Self-host fonts (kill the external `piqigroup.com` `@font-face` fetch), `sizes`/`priority` audit, full reduced-motion + contrast sweep.

## Non-negotiables carried through every phase

- Plain CSS in `globals.css` only — no Tailwind, no CSS-in-JS, no per-component modules.
- Dark theme only — no `[data-theme="light"]`, no toggle.
- PIQI red `#bb181d` as editorial statement; red is the only hue; accent capped at 2–4 statement uses per page.
- Group pages (Home/About/Contact) stay abstract/architectural/motif only — never vertical imagery.
- All 16 routes stay `○ Static` (consulting spokes `● SSG`); `validate-schema` reports ≥66 schemas across 16 routes.
- No real copy / no fabricated facts; lorem placeholders only; schema-bound fields keep real strings or stay omitted (never lorem).
- Quality gate (`build` / `lint` / `typecheck` / `validate-schema`) green after every phase.

## Things that REQUIRE USER APPROVAL before building

Quarantined in [`01-REDESIGN-PLAN.md` §6](01-REDESIGN-PLAN.md): any WebGL/Three.js/R3F/GPGPU/canvas-shader hero; a lightweight canvas/OGL fragment-shader; changing the typeface family; any second hue; UI sound design; and the first-visit preloader as a permanent feature. Default posture declines all of these — the pure-CSS path achieves the signature look without them.
