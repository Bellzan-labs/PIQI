# PIQI competitor audit — summary

This folder contains one competitor-audit file per PIQI vertical (consulting, property, fashion, yachts, auto, coaching). Each file surveys 4–6 live competitor websites, records verbatim positioning and copy where useful, draws out patterns, and closes with PIQI-specific recommendations split into what we can write now versus what needs client input. Competitors were selected from the brief's candidate list, with substitutions made when sites returned 403s or refused connections — McKinsey (socket close), PwC ZA (403), Bowmans (403), BMW ZA (timeout), Dream Yacht (403 with redirects), Gert-Johan Coetzee and Orka (ECONNREFUSED); substitutes were drawn from the same vertical (Bain, EY ZA, KPMG ZA, BMW global, Atlantis Yacht Charters, Catamaran Charter, Auto Magic, Midas, Elbeth Gillis, Maryke, Casey Jeanne).

## Cross-vertical observations
- **Nobody in any of PIQI's six markets publishes real pricing for a core service.** Even partial engagement-shape clarity would be a differentiator in every vertical.
- **Process transparency substitutes for price transparency.** The competitors that do best — Auto Magic's 24-step diagram, The Moorings' calendar-level pricing, CoachHub's 5% acceptance stat — answer the buyer's real question ("what happens, and can I trust it?") through concreteness rather than adjectives.
- **Every vertical has a rhythm problem PIQI can name.** Lead times for bespoke gowns, turnaround days for panel beating, cadence for coaching, response time for maintenance, trip-length reality for charters, FEL duration on commercial projects. PIQI is small enough to be honest about time in a way scaled competitors are not.
- **Multi-vertical framing is unique.** Nobody we surveyed runs consulting + property + fashion + yachts + auto + coaching under one roof, so PIQI's group architecture is itself a trust signal if presented as "one accountable team, six specialisms."

## Top 3 ranked recommendations
1. **Add a "rhythm and response" line to every vertical page.** One sentence per service describing how long things typically take and who stays in contact — no invented numbers, just the honest shape. This is the single strongest cross-vertical move.
2. **Use process sequences as the default proof structure.** Swap generic "why choose us" blocks for "what happens when you work with us" sequences (borrowed most strongly from Auto Magic and the Big Four case-study model).
3. **Carry a consistent practitioner voice across all six verticals** — plain, specific, operator-to-operator. It is the voice most competitors in each vertical skip, and it is the one PIQI can naturally hold because the group is small.

Cross-reference: each vertical file's "Recommendations for PIQI's [vertical] page" section is written to be pasted directly into content-work tickets without further research.
